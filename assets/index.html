HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element is what you see in the tab the page is in.
  • The h1 element is the main heading of the page.
  • The p element is used to create a paragraph.
  • The img element is used to add an image to the page.
  • The a element is used to create a hyperlink.
  • The ul element is used to create an unordered list.
  • The ol element is used to create an ordered list.
  • The li element is used to create a list item.
  • The table element is used to create a table.
  • The tr element is used to create a table row.
  • The td element is used to create a table cell.
  • The th element is used to create a table header.
  • The form element is used to create a form.
  • The input element is used to create an input field.
  • The label element is used to create a label for an input field.
  • The button element is used to create a button.
  • The div element is used to create a division or section in an HTML document.
  • The span element is used to create an inline section in an HTML document.
  • The id attribute is used to uniquely identify an element.
  • The class attribute is used to specify a class for an element.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A border is a line that goes around the outside of an element.
  • The box-sizing property tells the browser how to calculate the total width and height of an element.
  • The display property tells the browser how to display an element.
  • The position property tells the browser how to position an element.
  • The float property tells the browser how to float an element.
  • The clear property tells the browser which sides of an element it is allowed to float next to.
  • The overflow property tells the browser what to do if the content inside an element is too big.
  • The z-index property tells the browser which elements should be on top of other elements.
  • The background-color property tells the browser what color to make the background of an element.
  • The color property tells the browser what color to make the text inside an element.
  • The font-family property tells the browser what font to use for the text inside an element.
  • The font-size property tells the browser what size to make the text inside an element.
  • The font-weight property tells the browser how bold to make the text inside an element.
  • The text-align property tells the browser how to align the text inside an element.
  • The text-decoration property tells the browser how to decorate the text inside an element.
  • The text-transform property tells the browser how to transform the text inside an element.
  • The transition property tells the browser how to transition an element.
  • The transform property tells the browser how to transform an element.
  • The animation property tells the browser how to animate an element.
  • The @media rule tells the browser how to style an element at a certain screen size.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add .: adds all files to the staging area
  • git commit -m "message": commits the files to the local repository
  • git push origin branch-name: pushes the files to the remote repository
  • git pull origin branch-name: pulls the files from the remote repository
  • git checkout branch-name: switches to the specified branch
  • git merge branch-name: merges the specified branch with the current branch
  • git branch -d branch-name: deletes the specified branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Functions are reusable blocks of code that perform a specific task.
  • Parameters are variables that are used to store the data that is passed into a function.
  • Arguments are the actual data that is passed into a function.
  • Arrays are a data structure that allows us to store multiple values in a single variable.
  • Objects are a data structure that allows us to store multiple values in a single variable.
  • Loops are a programming tool that allows us to repeat a block of code.
  • Conditionals are a programming tool that allows us to execute code based on a condition.
  • Operators are used to perform operations on variables and values.
  • Comparison operators are used to compare two values.
  • Logical operators are used to determine the logic between variables or values.
  • Truthy values are values that are considered true when evaluated in a Boolean context.
  • Falsy values are values that are considered false when evaluated in a Boolean context.
  • Truthy and falsy values are values that are considered true or false when evaluated in a Boolean context.
